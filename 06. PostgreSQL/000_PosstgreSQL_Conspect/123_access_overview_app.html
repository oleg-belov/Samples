Создание ролей 

=> CREATE ROLE librarian LOGIN PASSWORD 'librarian';

CREATE ROLE

=> CREATE ROLE storekeeper LOGIN PASSWORD 'storekeeper';

CREATE ROLE

=> CREATE ROLE buyer LOGIN PASSWORD 'buyer';

CREATE ROLE

Настройки подключения

Настройки по умолчанию разрешают подключение с локального адреса по паролю. Нас это устраивает.

Привилегии public

У роли public надо отозвать лишние привилегии. 

=> REVOKE EXECUTE ON ALL FUNCTIONS IN SCHEMA bookstore FROM public;

REVOKE

=> REVOKE CONNECT ON DATABASE bookstore FROM public;

REVOKE

Привилегии покупателя

Покупатель должен иметь доступ к поиску книг и их покупке. 

=> GRANT CONNECT ON DATABASE bookstore TO buyer;

GRANT

=> GRANT USAGE ON SCHEMA bookstore TO buyer;

GRANT

=> GRANT EXECUTE ON FUNCTION get_catalog(text,text,boolean) TO buyer;

GRANT

=> GRANT EXECUTE ON FUNCTION buy_book(integer) TO buyer;

GRANT

Привилегии кладовщика

Кладовщик должен иметь доступ к каталогу для заказа книг. 

=> GRANT CONNECT ON DATABASE bookstore TO storekeeper;

GRANT

=> GRANT USAGE ON SCHEMA bookstore TO storekeeper;

GRANT

=> GRANT SELECT,UPDATE(onhand_qty) ON catalog_v TO storekeeper;

GRANT

=> GRANT EXECUTE ON FUNCTION book_name(integer,text,integer) TO storekeeper;

GRANT

=> GRANT EXECUTE ON FUNCTION authors(books) TO storekeeper;

GRANT

Привилегии библиотекаря

Библиотекарь должен иметь доступ к просмотру и добавлению книг и авторов. 

=> GRANT CONNECT ON DATABASE bookstore TO librarian;

GRANT

=> GRANT USAGE ON SCHEMA bookstore TO librarian;

GRANT

=> GRANT SELECT ON catalog_v TO librarian;

GRANT

=> GRANT SELECT ON authors_v TO librarian;

GRANT

GRANT
